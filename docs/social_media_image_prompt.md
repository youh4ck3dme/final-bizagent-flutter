# Modern Social Media Sharing Image Design Prompt

**Purpose:** Replace the current og_image.png with a more modern, engaging social media sharing image for BizAgent PWA.

**Technical Requirements:**
- Dimensions: 1200 x 630 pixels (optimal for Facebook, Twitter, LinkedIn)
- Format: PNG with transparent background support
- File size: <1MB for fast loading
- Style: Modern, professional, tech-forward

**Design Concept:**
Create a sleek, modern image that captures BizAgent's AI-powered business management essence.

**Visual Elements:**

1. **Background:**
   - Clean white or very light gray (#F8FAFC) background
   - Subtle geometric pattern or soft gradients in Slovak blue (#0038A8) tones
   - No busy patterns - keep it clean and professional

2. **Main Visual:**
   - Central focus: Modern invoice document with AI elements
   - Invoice appears to be "materializing" or being generated by AI
   - Include subtle AI/tech elements: neural network patterns, data flow lines, or floating geometric shapes

3. **Typography & Branding:**
   - **Primary Headline:** "BizAgent" in bold, modern font (similar to Inter or Outfit)
   - **Subtitle:** "Inteligentný Business Asistent" in Slovak
   - **Tagline:** "AI-powered fakturácia pre moderné firmy"
   - Use Slovak blue (#0038A8) for primary text, red accents (#D0021B) for highlights

4. **Key Features Highlight:**
   - Small icons/text for: "AI Faktúry", "Automatické výdavky", "Real-time prehľady"
   - Positioned elegantly around the main visual

5. **Style & Mood:**
   - **Color Palette:** Slovak flag inspired (blue #0038A8, red #D0021B, white)
   - **Aesthetic:** Glassmorphism elements, subtle shadows, clean lines
   - **Tech Feel:** Modern SaaS app look, professional but approachable
   - **Trust Elements:** Subtle checkmarks or verification badges

**AI Image Generation Prompt:**

```
Create a modern social media sharing image for BizAgent, an AI-powered business assistant app. The image should be 1200x630 pixels.

MAIN SUBJECT: A sleek, professional invoice document appearing to materialize from AI technology, floating in a clean white space.

VISUAL STYLE: Modern glassmorphism design with subtle blue gradients. Clean, minimal, tech-forward aesthetic. Professional SaaS application look.

COLOR SCHEME: Primary - Slovak blue (#0038A8), Accent - Slovak red (#D0021B), Background - Clean white (#FFFFFF) with subtle geometric patterns.

CONTENT ELEMENTS:
- Central: Semi-transparent invoice with modern typography
- Top-left: "BizAgent" logo in bold modern font
- Top-right: Small AI brain icon with subtle glow
- Bottom: Feature highlights - "AI Faktúry • Automatické výdavky • Real-time prehľady"
- Subtle geometric shapes and data flow lines in background

TYPOGRAPHY: Clean sans-serif fonts, hierarchy with bold headlines and readable subtitles.

ATMOSPHERE: Trustworthy, innovative, professional. Conveys intelligence and ease of use.

FINAL OUTPUT: High-resolution PNG, optimized for social media sharing.
```

**Implementation Steps:**

1. Generate the image using AI tools (Midjourney, DALL-E, Stable Diffusion, etc.)
2. Save as `og_image_modern.png` in `BizAgent-main/web/assets/`
3. Update `index.html` meta tags to point to the new image
4. Test the sharing preview on social media platforms

**File Location:** `BizAgent-main/web/assets/og_image_modern.png`

**Meta Tags to Update in index.html:**
```html
<meta property="og:image" content="assets/og_image_modern.png">
<meta property="twitter:image" content="assets/og_image_modern.png">
```

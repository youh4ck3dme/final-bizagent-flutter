## ğŸ¯ Best Practices

### DO:
- Use high-quality images
- Keep text minimal and impactful
- Use brand colors consistently
- Make key message obvious
- Show actual app UI if possible
- Test readability on mobile

### DON'T:
- Overcrowd with text
- Use low-resolution images
- Include misleading claims
- Use generic stock photos
- Make text too small
- Violate brand guidelines
- Use competitor names or logos

## ğŸ“ Text Guidelines

### Font Recommendations:
- **Primary:** Inter, SF Pro, Roboto (clean sans-serif)
- **Size:** Minimum 40px for main text
- **Weight:** Bold or Semi-bold for headlines
- **Color:** White or light gray on dark background

### Text Hierarchy:
1. **Primary message:** Large, bold (60-80px)
2. **Secondary message:** Medium (30-40px)
3. **Tertiary/features:** Small (20-30px)

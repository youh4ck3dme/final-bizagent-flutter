## ğŸ¯ Asset Requirements Summary

### 1. App Icon
- **Size:** 512 x 512 px
- **Format:** 32-bit PNG with alpha channel
- **Max file size:** 1024 KB
- **Location:** `icons/`
- **Guide:** See `icons/ICON_DESIGN_GUIDE.md`

### 2. Feature Graphic
- **Size:** 1024 x 500 px
- **Format:** PNG or JPEG
- **Max file size:** 1 MB
- **Location:** `feature_graphic/`
- **Guide:** See `feature_graphic/FEATURE_GRAPHIC_GUIDE.md`

### 3. Screenshots
- **Phone:** Minimum 2, maximum 8
- **Size:** 1080 x 1920 px (recommended)
- **Format:** JPEG or 24-bit PNG
- **Location:** `screenshots/phone/`
- **Guide:** See `screenshots/SCREENSHOTS_GUIDE.md`

### 4. Store Listing Text
- **Title:** Max 30 characters
- **Short description:** Max 80 characters
- **Full description:** Max 4000 characters
- **Location:** `store_listings/sk_SK/` and `store_listings/en_US/`

# Icon Design Guide - BizAgent

## üì± Google Play Store Icon Requirements

### Icon Specifications

**High-resolution icon (required):**
- **Size:** 512 x 512 px
- **Format:** 32-bit PNG (with alpha channel)
- **Maximum file size:** 1024 KB
- **Shape:** Square (full-bleed)

**Important Notes:**
- Google Play will automatically apply the mask shape
- Do not round corners yourself
- Do not add shadows (system adds them)
- Ensure icon looks good on different backgrounds

## üéØ Best Practices

### DO:
- Use simple, bold shapes
- Ensure high contrast
- Make it distinctive and memorable
- Use brand colors consistently
- Test at multiple sizes
- Keep it timeless

### DON'T:
- Use photos or complex images
- Add text labels
- Use generic stock icons
- Copy competitor designs
- Make it too detailed
- Use too many colors

## üìê Icon Adaptive Design

For Android Adaptive Icons, consider:
- **Safe zone:** Keep important elements in center 66% circle
- **Foreground layer:** Main icon elements
- **Background layer:** Can extend to full canvas
- Icons will be masked to different shapes (circle, squircle, rounded square)
